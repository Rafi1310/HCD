# HCD
Mijn repository voor het vak HCD

## Testpersoon
De testpersoon waar ik een website voor ga maken is Darice. 
Darice wil graag de inhoud van podcasts kunnen volgen, inclusief alle hoorbare details zoals sfeergeluiden, geluidseffecten, en nuances zoals intonatie en andere eigenschappen van de stemmen. Is de persoon bijvoorbeeld boos? Verdrietig? Vrolijk?

Ze wil films kunnen volgen, ook de hoorbare details zoals geluidseffecten, (spannende) muziek, etc.

## Mijn opdracht
Ik ga er voor zorgen dat Darice beter films kan volgen en nog steeds dezelfde ervaring heeft zoals mensen die wel kunnen horen.
Ik heb alvast wat onderzoek gedaan naar de persoon en op haar persoonlijke website zegt ze "Voor communicatie in de horend wereld zal schrijftolken (ondertiteling) altijd mijn voorkeur hebben". Ik ga dus zorgen dat er goede ondertiteling bij zit die je ook kan aanpassen naar persoonlijke voorkeur. Ik heb hiervoor de volgende ideeen.

- Duidelijke ondertiteling
- Ondertiteling groter en kleiner kunnen maken
- Met kleur werken voor emoties. (bijv. rood voor boos, groen voor blij, etc.)
- Iconen gebruiken in ondertiteling
- Verschillende soorten lettertypen gebruiken voor verschillende emoties of "vibes"

Voor de rest heb ik ook nog andere ideeen wat het film maken voor doven een stuk interessanter maakt. 
- Het scherm niet statisch maken, misschien een beetje beweging erin
- Lichtflitsen erin, als er bijv. donder is dan zie je een flits.
- Iets van animaties erin gooien
